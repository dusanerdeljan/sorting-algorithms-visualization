# sorting-algorithms-visualization

Sorting algorithms visualization made using olcPixelGameEngine (Visual Studio 2015 Project)

Currently supported algorithms:

 * Bubble sort
 * Selection sort
 * Insertion sort
 * Merge sort
 * Quick sort
 * Heap sort
